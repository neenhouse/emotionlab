# EmotionLab

Emotional intelligence development platform for children ages 5-15. Develops empathy, self-regulation, conflict resolution, and social awareness through interactive exercises.

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite 8
- **Styling**: CSS custom properties (violet/pink/teal design system)
- **Deploy**: Cloudflare Pages
- **Testing**: Vitest + React Testing Library
- **Tooling**: pnpm (package manager), mise (runtime versions)

## Commands

```bash
pnpm dev        # Start dev server
pnpm build      # TypeScript check + Vite build
pnpm test       # Run Vitest
pnpm lint       # ESLint
```

## Design Identity

- Primary: Warm violet (#8B5CF6)
- Secondary: Soft pink (#EC4899)
- Accent: Gentle teal (#14B8A6)
- Light, playful but not childish -- premium children's education
- Rounded corners, soft shadows, gentle gradients
- Safe, encouraging, growth-oriented
